# منصة مدرس واحد 🎓

منصة تعليمية تفاعلية مصممة خصيصًا لمدرس واحد، بهدف عرض الكورسات، الدروس، الاختبارات، وبيانات الطلاب بشكل منظم واحترافي، مع تجربة استخدام بسيطة وسريعة للطالب والمدرس.

---

## 📌 فكرة المشروع

المشروع عبارة عن منصة تعليمية Frontend كاملة يمكن استخدامها كنموذج أولي أو كأساس لمنصة تعليمية حقيقية لمدرس واحد.

توفر المنصة واجهات عامة للزائر، ولوحات تحكم للطالب والمدرس، مع صفحات للكورسات، الصفوف الدراسية، التواصل، وعن المنصة.

---

## ✨ المميزات

### للزائر

- صفحة رئيسية احترافية بتصميم تعليمي جذاب.
- عرض مميزات المنصة بطريقة واضحة.
- صفحة الصفوف الدراسية: أولى، ثانية، ثالثة ثانوي.
- صفحة عن المنصة لعرض هوية المدرس والمنصة.
- صفحة تواصل تحتوي على بيانات التواصل وروابط مهمة.
- تصميم Responsive مناسب للموبايل والتابلت والديسكتوب.

### للطالب

- تصفح الكورسات المتاحة.
- عرض تفاصيل الكورس والدروس.
- متابعة المحتوى التعليمي.
- تجربة استخدام سهلة وواضحة.

### للمدرس / الإدارة

- هيكل جاهز لإدارة المحتوى التعليمي.
- إمكانية التوسع لاحقًا لإضافة لوحة تحكم كاملة.
- تصميم قابل للتطوير والربط مع Backend حقيقي.

---

## 🛠️ التقنيات المستخدمة

## 🚀 تشغيل المشروع محليًا

### 1. تحميل الاعتمادات

```bash
npm install
```

### 2. تشغيل المشروع

```bash
npm run dev
```

بعد التشغيل، افتح الرابط الذي يظهر في التيرمنال، غالبًا:

```bash
http://localhost:5173
```
